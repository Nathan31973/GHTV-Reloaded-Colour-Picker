# GHTV-Reloaded-Mod-Tools
This Simple Tool Modify Guitar Hero Live and GHTV:Reloaded

Feature
- Custom Colour for Highway FX
- HyperSpeed for all songs (As of 15/12/2022)

Controls:
- Go back a menu / Quit(on main menu): Escape Key
